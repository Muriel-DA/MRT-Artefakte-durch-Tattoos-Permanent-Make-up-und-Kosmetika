# MRT-Artefakte-durch-Tattoos-Permanent-Make-up-und-Kosmetika
In dieser Arbeit wurde untersucht, inwiefern Tattoos, PMU und kosmetische Produkte Artefakte in der MRT verursachen. Mithilfe eines Phantom-Setups und In-vivo-Messungen zeigte sich, dass Artefakte produktspezifisch auftreten und insbesondere bei wasserfestem Augen-Make-up relevant sind.
